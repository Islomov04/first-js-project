# ToDo List Project

Простое приложение для управления списком дел.  
Можно добавлять задачи, отмечать выполненные и удалять их.  

##  Демо
https://github.com/Islomov04/first-js-project 

##  Функции
- Добавление новой задачи
- Удаление задачи
- Отметка выполнения
- Сохранение задач в LocalStorage

##  Технологии
- HTML
- CSS
- JavaScript
